# MediQueue Frontend

MediQueue is a tutor booking platform built with Next.js.  
Students can browse tutors, book sessions, and manage their bookings easily.

## Features

- Google & Email Login
- Protected Routes
- Add Tutor
- My Tutors
- My Bookings
- Responsive Design

## Tech Stack

- Next.js
- Tailwind CSS
- DaisyUI
- Better Auth
- Axios

## Run Locally

```bash
npm install
npm run dev
```

## Environment Variables

```env
NEXT_PUBLIC_SERVER_URL=http://localhost:5000
```
